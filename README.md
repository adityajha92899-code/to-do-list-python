# To-Do List App (Python)

This is a simple console-based To-Do List application made using Python.  
It can add tasks, show all tasks, and delete tasks.

## How to Run
- Open terminal
- Type: `python to-do.py`

## Features
- Add tasks
- View tasks
- Delete tasks
- Simple menu-based UI

## Project Purpose
This project was created for the “Tools for Programming, Learning, and Collaboration” assignment using VS Code, Git, and GitHub.
